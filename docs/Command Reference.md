# Command Reference

Every `inflights` command grouped by domain.

---

## Authentication

| Command | Description |
|---------|-------------|
| [[inflights login]] | Authenticate via browser |
| [[inflights logout]] | Clear local token |
| [[inflights whoami]] | Show current user and role |

## Quotes

| Command | Description |
|---------|-------------|
| [[inflights quote assign]] | Assign a quote to a user for review |
| [[inflights quotes]] | List quotes assigned to you |
| [[inflights quote show]] | Show details of a specific quote |
| [[inflights quote confirm]] | Confirm / accept a quote |

## Flights

| Command | Description |
|---------|-------------|
| [[inflights flights]] | List flights (filterable) |
| [[inflights flight show]] | Show flight details |
| [[inflights flight add]] | Register an external flight |
| [[inflights flight status]] | Update a flight's status |
| [[inflights flight assign-pilot]] | Assign a pilot to a flight |
| [[inflights flight assign-processor]] | Assign a processor to a flight |

## Data & Uploads

| Command | Description |
|---------|-------------|
| [[inflights upload images]] | Upload drone images for a flight |
| [[inflights upload data]] | Upload processed data for a flight |
| [[inflights download]] | Download data for a flight |

## Equipment

| Command | Description |
|---------|-------------|
| [[inflights gear list]] | List your registered equipment |
| [[inflights gear add]] | Register new equipment |

## Services

| Command | Description |
|---------|-------------|
| [[inflights services]] | List available services |
| [[inflights order]] | Order a service |
