# Command Reference

Every `inflights` command grouped by domain.

---

## Authentication

| Command | Description |
|---------|-------------|
| [[inflights login]] | Authenticate via browser |
| [[inflights logout]] | Clear local token |
| [[inflights whoami]] | Show current user and role |

## Quotes

| Command | Description |
|---------|-------------|
| [[inflights quotes]] | List quotes assigned to you |
| [[inflights quote show]] | Show details of a specific quote |
| [[inflights quote confirm]] | Confirm / accept a quote or estimate |
| [[inflights quote reject]] | Reject a quote or estimate |

## Flights

| Command | Description |
|---------|-------------|
| [[inflights flights]] | List flights (filterable by status, public UID) |
| [[inflights flight show]] | Show flight details and deliverable types |
| [[inflights order]] | Order a new flight with GeoJSON area |

## Proposals

| Command | Description |
|---------|-------------|
| [[inflights proposal list]] | List proposals assigned to you |
| [[inflights proposal show]] | Show proposal details |
| [[inflights proposal accept]] | Accept a flight proposal |
| [[inflights proposal reject]] | Reject a flight proposal |

## Data & Uploads

| Command | Description |
|---------|-------------|
| [[inflights upload images]] | Upload drone images for a flight |
| [[inflights upload data]] | Upload processed deliverables for a flight |
| [[inflights download]] | Download data for a flight |

## Equipment

| Command | Description |
|---------|-------------|
| [[inflights gear list]] | List available equipment on the platform |
| [[inflights gear mine]] | List equipment on your profile |
| [[inflights gear add]] | Add equipment to your profile |
| [[inflights gear remove]] | Remove equipment from your profile |

## Services

| Command | Description |
|---------|-------------|
| [[inflights services]] | List available services |
