# Customer

A customer needs drone data for a specific area. They request quotes, confirm orders, and download deliverables.

## Key commands

- [[inflights quote request]] — request a quote for an area + insight type
- [[inflights quotes]] — check on pending quotes
- [[inflights quote confirm]] — accept a quote (creates a flight)
- [[inflights flights]] — see flights tied to your account
- [[inflights flight show]] — check status, see who's assigned
- [[inflights download]] — download processed deliverables
- [[inflights services]] — browse available services
- [[inflights order]] — order a service directly
